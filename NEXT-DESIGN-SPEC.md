## 一、项目概述

**网站名称：** Next
**中文标语：** 今晚看什么？
**核心定位：** 帮用户解决"选择困难症"的影剧推荐工具
**访问地址：** http://localhost:8651
**目标用户：** 25-40岁，不知道该看什么电影/电视剧的普通观众
**核心功能：** 本周热榜 / 猜你喜欢（AI推荐） / 高分推荐 / 类型筛选 / 影剧详情 / 收藏与记录

**主要参考网站：**
- [apple.com](https://www.apple.com) — 精致、留白、产品感
- [themoviedb.org](https://www.themoviedb.org) — 影剧数据展示
- [大米星球](https://www.dmq8a2x9s1.shop) — 国内影剧站布局

## 三、设计系统

### 3.1 颜色

| 用途 | 色值 | 说明 |
|:----|:-----|:-----|
| 背景 | `#0A0A0F` | 深沉黑，微暖 |
| 卡片底色 | `#1C1C2E` | 微微泛紫 |
| 主文字 | `#F0F0F0` | 柔和白 |
| 辅助文字 | `#6E6E73` | 灰色 |
| 强调色 | `#D4A855` | 暖金色 |
| 强调渐变 | `linear-gradient(135deg, #D4A855, #B8923E)` | 按钮渐变 |
| 评分色 | `#01D277` | TMDB风格绿色 |
| 喜欢色 | `#E85D6F` | 柔和红 |
| 边框 | `rgba(255,255,255,0.06)` | 极淡白 |

### 3.2 字体

| 层级 | 字号 | 字重 | 行高 | 字族 |
|:----|:----|:----|:----|:-----|
| Hero标题 | 48px(桌面)/32px(手机) | 700 | 1.05 | Playfair Display |
| 区块标题 | 24px | 700 | 1.2 | PingFang SC / Inter |
| 卡片标题 | 14px | 500 | 1.3 | PingFang SC / Inter |
| 正文 | 14-16px | 400 | 1.5 | PingFang SC / Inter |
| 辅助文字 | 12-14px | 400 | 1.4 | PingFang SC / Inter |
| 标签 | 12px | 600 | 1.2 | PingFang SC / Inter |

**字族优先级：**
- 中文：`PingFang SC` → `Noto Sans SC` → 系统默认
- 英文：`Inter` → `SF Pro` → `Helvetica Neue`
- 标题衬线：`Playfair Display` → 衬线体系统默认

### 3.3 圆角

| 级别 | 值 | 用途 |
|:----|:---|:-----|
| --r-sm | 4px | 标签徽章 |

| --r-xl | 20px | 推荐卡 / 弹窗 |
| --r-pill | 999px | CTA按钮 / 头像 |

## 四、组件规格

### 4.1 Hero 推荐区

| 属性 | 值 |
|:----|:----|
| 高度 | 70vh（min 420px, max 600px） |
| 背景 | 全幅图片，暗色渐变叠加（从底部深色延伸到顶部半透明） |
| 渐变 | `linear-gradient(to top, #0A0A0F 0%, #0A0A0F 15%, rgba(10,10,15,0.6) 40%, rgba(10,10,15,0.3) 70%, rgba(10,10,15,0.5) 100%)` |
| 标题阴影 | `text-shadow: 0 2px 30px rgba(0,0,0,0.8)` |
| 按钮样式 | 药丸形（border-radius: 999px）, 渐变背景, 悬停 scale(1.02) |

### 4.2 影剧卡片

| 属性 | 值 |
|:----|:----|
| 宽高比 | 2:3（竖版海报） |
| 圆角 | 14px |
| 阴影 | 默认 `0 2px 8px rgba(0,0,0,0.3)`，悬停加深+上移4px |
| 悬停 | `translateY(-4px)` + 海报 scale(1.05) |
| 评分徽章 | 右上角，绿底半透明，悬停显示 |

### 4.3 评分徽章

| 属性 | 值 |
|:----|:----|
| 背景 | `rgba(1, 210, 119, 0.15)` |
| 文字 | `#01D277` |
| 圆角 | 4px |
| 字号 | 12px, 600字重 |
| 图标 | ★ + 数字 |

### 4.4 药丸按钮（CTA）

| 属性 | 值 |
|:----|:----|
| 形状 | `border-radius: 999px` |
| 渐变 | `linear-gradient(135deg, #D4A855, #B8923E)` |
| 内边距 | 12px 28px（标准） |
| 字号 | 14px, 600字重 |
| 悬停 | `scale(1.02)`, `filter: brightness(1.1)` |
| 点击 | `scale(0.98)` |

### 4.5 横向滚动区

| 属性 | 值 |
|:----|:----|
| 溢出 | `overflow-x: auto` |
| 吸附 | `scroll-snap-type: x mandatory` |
| 子项吸附 | `scroll-snap-align: start` |
| 滚动条 | 隐藏（保留功能） |
| 间距 | `gap: 16px` |

## 五、交互规范

| 交互 | 行为 | 时长 |
|:----|:----|:----|
| 卡片悬停 | 上移4px + 阴影加深 + 海报放大1.05x | 0.3s ease-out |
| 按钮悬停 | 放大1.02 + 亮度1.1 | 0.3s ease-out |
| 按钮点击 | 缩小至0.98 | 0.1s |
| 页面进场 | 淡入 + 上浮24px | 0.6s ease-out |
| 区块滚动入视口 | 渐显上浮 | 0.6s ease-out |
| 分类切换 | 即时替换，无刷新 | 即时 |
| Tab切换 | 路由跳转，无页面刷新 | 即时 |
